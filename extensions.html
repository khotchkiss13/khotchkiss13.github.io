The Lego NXT Package created by Conner Hudson is a block module and local server for Snap to interface with the Mindstorms NXT. Check out the GitHub repository here to get started!

The Nintendo Wiimote Package developed by Connor Hudson allows users to use the motion controller from the very popular Nintendo Wii video game console as an input device for Snap! To begin using this, chack out the GitHub repository here!

The Parallax S2 robot package by Connor Hudson allows for users to program the Scribbler 2 robot by using Snap! Check out the GitHub Repository here to get started!

With the LEAP Motion extension developed by Connor Hudson, users can now utilize the impressive LEAP Motion motion detection as an input device in Snap! Check out the GitHub repository here to start!

Speech Synthesis

With his Speech Synthesis package, Connor Hudson has given Snap! the ability to talk back to its users! Check out the GitHub repository here to enable this extension!

Arduino Package 1

With this Arduino Package by Alan Yorinks, users can now program Arduinos by using Snap!, allowing for an arduino user to never have to write a bit of Arduino sketch code! Check out the GitHub repository here to get started!

Arduino Package 2

With this Arduino Package developed by Bernat Romagosa and Citilab, users are greeted with a modified version of Snap! that is made specifically for use with Arduino microcontrollers! Find out more here to get started!

The Finch and Hummingbird Robotics Package for Snap! created by Tom Lauwers allows for users to use Snap! to program the Finch Robot and Hummingbird Arduino using specialized blocks made just for these devices. Check out how to start working here at the Hummingbird kit website!
